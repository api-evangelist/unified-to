The data an AI agent needs is often sitting in the customer's warehouse: ops metrics in Snowflake, analytics in BigQuery, and application data in Supabase. This guide explains how the Unified Datastore API normalizes four operations across those warehouses through a single interface — reading records, querying with filters or SQL, writing results back, and reacting to data changes.
